# SHUFOO掲載用CSV・画像ZIP作成ツール 使い方マニュアル

SHUFOO（チラシ掲載サービス）へ提出する「店舗ごとの登録CSV」と「画像ZIP」を、
画面の質問に答えるだけで自動的に作成するツールです。
前回分のCSVを手作業でコピーして日付やタイトルを書き換える必要はもうありません。
このマニュアルの通りに操作すれば使えるようになっています。

---

## 1. このツールが行うこと

1. 「お買い得チラシ」「やすい！」のどちらのパターンかを選ぶ
2. 掲載開始日・終了日、チラシタイトル、画像ファイル名を質問に答えて入力する
3. パターンに登録されている全店舗分の行を自動的に組み立てる
4. SHUFOO所定の形式（文字コードcp932）でCSVを保存する
5. CSVで使われている画像をまとめてZIPに保存する

**やること（あなたの作業）は「画像ファイルの用意」と「質問に答えること」だけです。**
SHUFOOの管理画面へのアップロード自体は、これまで通り手動で行ってください
（このツールはローカルにファイルを作るところまでが対象です）。

---

## 2. 初回だけ行う準備

### 準備1: 店舗マスター（`config.json`）を確認する

`日次業務/shufoo/config.json` に、パターンごとの対象店舗（店舗ID・店舗名）があらかじめ
登録されています。通常は編集不要です。

店舗の新規追加・削除・パターンの入れ替えがあったときだけ、このファイルを直接編集してください。
1店舗分は以下の形式です。`entry_id`はSHUFOO側の登録番号なので、新規店舗の場合は
本部・SHUFOO担当に確認してから入力してください。

```json
{"entry_id": "20250221", "store_id": "881827", "store_name": "お菓子のデパートよしや　○○店"}
```

編集後は、末尾以外の店舗情報の後ろに必ずカンマ `,` を付けてください（一番最後の店舗の後ろには付けません）。

### 準備2: Pythonが使えることを確認する

VS Codeのターミナルで以下を実行し、バージョンが表示されればOKです（追加インストールは不要です。標準機能のみで動作します）。

```powershell
python --version
```

## 3. 毎回の使い方

### 手順1: 画像ファイルを用意する

配信するチラシ画像（JPG）を `日次業務/shufoo/` フォルダの中に置いてください。
既存の画像を再利用する場合はこの手順は不要です。

### 手順2: ツールを実行する

```powershell
cd "C:\one_tenri\OneDrive\HTY\vscode\日次業務\shufoo"
python make_shufoo_csv.py
```

### 手順3: 質問に順番に答える

以下のような質問が表示されるので、順番に答えていきます。

| 質問 | 入力例 | 補足 |
|---|---|---|
| パターンを選択してください | `1` または `2` | 1: お買い得チラシ、2: やすい！ |
| 掲載開始日 | `7/10` | 年を省略すると今年になります |
| 掲載終了日 | `7/11` | 同上 |
| チラシタイトル | `お買い得チラシ7月2週目` | そのままCSVに使われます |
| 既定の画像ファイル名 | `7.10-11.JPG` | 手順1で置いたファイル名 |
| 画像が異なる店舗はありますか？ | 番号 or 空Enter | 天理店だけ画像が違う、などの例外があるときだけ番号を選ぶ。無ければ何も入力せずEnter |

**掲載開始日時・終了日時は自動計算されます**（開始日の前日19:00〜終了日15:00で固定）。
時刻を自分で入力する必要はありません。

例外店舗を選んだ場合は、続けてその店舗専用の画像ファイル名を聞かれます。
例外がなくなったら、番号を聞かれた画面で何も入力せずEnterを押すと次に進みます。

### 手順4: 結果を確認する

質問がすべて終わると、以下のファイルが自動生成されます（前回分は上書きされます）。

- `tirasi.csv` または `tirasiyasui.csv`（選んだパターンに応じて）
- `tirasi_images.zip` または `tirasiyasui_images.zip`

画面に「CSVを保存しました」「画像ZIPを保存しました」と件数が表示されれば成功です。
あとはこの2つのファイルをSHUFOOの管理画面から提出してください。

## 4. よくある質問・トラブル対応

### Q. 「警告: 以下の画像ファイルが見つかりません」と出る

指定した画像ファイル名が `日次業務/shufoo/` フォルダに存在しません。
ファイル名のスペルミス（大文字小文字・拡張子）がないか確認するか、`n` と入力して中止し、
画像を正しい場所に置いてからやり直してください。`y` と入力すると、見つかった画像だけで
ZIPを作り、CSVには入力した名前がそのまま入ります（提出前に必ず画像がある方を確認してください）。

### Q. エラーで「1〜◯の番号を入力してください」と表示され続ける

パターン選択や店舗選択で、数字以外の文字や一覧にない番号を入力しています。
表示されている番号の範囲内で入力し直してください。

### Q. エラーで「日付の形式が正しくありません」と表示される

掲載開始日・終了日の入力形式が `7/10` や `2026/7/10` のようになっていません。
スラッシュ `/` 区切りで、月/日（または年/月/日）の順に入力し直してください。

### Q. 実行すると「FileNotFoundError: ... config.json」と出る

`日次業務/shufoo` フォルダの中にいない状態でツールを実行しています。
ターミナルで `cd 日次業務/shufoo` を実行してから、もう一度試してください。

### Q. 実行すると英語のエラー（Traceback）が出て止まる

上記の想定内エラー以外の問題が起きています。エラーメッセージの内容をコピーして、
次の作業依頼のときに一緒に伝えてください。

## 5. 覚えておいてほしいこと

- **同じファイル名で毎回上書きされます**: `tirasi.csv`等は実行するたびに前回分が上書きされます。前回分を残しておきたい場合は、実行前に別名でコピーを取ってください
- **SHUFOOへの提出は手動です**: このツールはローカルにCSV・ZIPを作るところまでです。SHUFOOの管理画面へのアップロードは、これまで通りご自身で行ってください
- **店舗の追加・削除は`config.json`を直接編集**: 対応方法は「2. 初回だけ行う準備」の準備1を参照してください
- **掲載時刻は前日19:00〜終了日15:00で固定**: 個別に時刻を変えたい場合は、次の作業依頼のときに相談してください
