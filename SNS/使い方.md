# SNS（TikTok）投稿分析ツール 使い方マニュアル

TikTokで人気の駄菓子・お菓子関連の動画を分析して、投稿のアイデアをもらうためのツールです。
このマニュアルの通りに操作すれば使えるようになっています。

---

## 1. このツールが行うこと

1. あなたがTikTokで見つけた気になる動画のURLを、いくつか登録しておく
2. ツールがその動画の「いいね数」「再生数」「キャプション」「ハッシュタグ」などを自動で調べる
3. 集めたデータから「よく使われるハッシュタグ」「反応が良かった投稿」などの傾向をまとめる
4. AI（Claude）が、その傾向をもとに「こんな動画を作ってみては？」という具体案を考えてくれる
5. すべての結果を1つのExcelファイルにまとめて出力する

**やること（あなたの作業）は「動画URLをメモしておく」ことと「実行する」ことだけです。**

---

## 2. 初回だけ行う準備

### 準備1: Claude APIキーを取得する

「Claude API」はAI（Claude）を使うための鍵（パスワードのようなもの）です。まだ持っていない場合は取得が必要です。取得方法が分からない場合は、次に作業するときに聞いてください。

キーを取得できたら、以下のファイルを開いてください。

```
SNS/config.json
```

`"anthropic_api_key"` の部分の `"ここにClaude APIキーを入力してください"` を、取得したキーに書き換えて保存します。

```json
{
  "video_urls": [],
  "wait_seconds_between_requests": 5,
  "headless": true,
  "anthropic_api_key": "ここに取得したキーを貼り付ける"
}
```

> ⚠️ **注意**: `config.json` にはキー（パスワード）が入るため、絶対に人に見せたりGitHub等にアップロードしたりしないでください（このツールでは自動的にGit管理から除外される設定になっているので、通常の操作をしている限り心配いりません）。

### 準備2: 必要な部品（パッケージ）をインストールする

VS Codeのターミナルを開いて、以下を実行してください（1回だけでOKです）。

```powershell
pip install -r SNS/requirements.txt
```

「Successfully installed ...」のような表示が出れば完了です。

## 3. 毎回の使い方（動画を分析したいとき）

### 手順1: 気になる動画のURLをコピーする

TikTokアプリやサイトで、駄菓子・お菓子関連の人気動画を見つけたら、その動画のURLをコピーします。

- アプリの場合: 動画の「共有」ボタン → 「リンクをコピー」
- URLはこんな形になっています: `https://www.tiktok.com/@ユーザー名/video/1234567890123456789`

### 手順2: URLを`config.json`に貼り付ける

`SNS/config.json` を開き、`"video_urls"` の `[ ]` の中にURLを貼り付けます。複数個登録できます（カンマ `,` で区切ってください）。

```json
{
  "video_urls": [
    "https://www.tiktok.com/@example1/video/1111111111111111111",
    "https://www.tiktok.com/@example2/video/2222222222222222222"
  ],
  "wait_seconds_between_requests": 5,
  "headless": true,
  "anthropic_api_key": "（自分のキー）"
}
```

保存を忘れずに（`Ctrl + S`）。

### 手順3: ツールを実行する

VS Codeのターミナルで以下を実行します（フォルダーへ移動してから実行する方法です）。

```powershell
cd "C:\one_tenri\OneDrive\HTY\vscode\SNS"
python run.py
```

フォルダー移動をせず、今いる場所から一気に実行したい場合は、以下のようにフルパスを指定しても動きます。

```powershell
python "C:\one_tenri\OneDrive\HTY\vscode\SNS\run.py"
```

どちらの方法で実行しても結果は同じです。

しばらくすると（動画1件あたり数秒〜十数秒かかります）、以下のようなメッセージが表示されて完了します。

```
指定されたURLから投稿データを収集しています...
2件の投稿データを取得しました: data\raw_20260704.csv
Claude APIで投稿企画案を生成しています...
レポートを出力しました: data\reports\20260704.xlsx
```

### 手順4: 結果（Excelレポート）を見る

`SNS/data/reports/` フォルダの中に、実行した日付のExcelファイル（例: `20260704.xlsx`）ができています。開いて確認してください。

- **「傾向データ」シート**: 集計した投稿数、よく使われるハッシュタグ、いいね数が多かった投稿の一覧（投稿日つき）
- **「投稿企画案」シート**: AIが考えた投稿アイデア

## 4. よくある質問・トラブル対応

### Q. 実行すると「config.jsonが見つかりません」と出る

`SNS`フォルダの中にいない状態でツールを実行しています。ターミナルで `cd SNS` を実行してから、もう一度 `python run.py` を試してください。

### Q. 実行すると「config.jsonの内容が正しいJSON形式ではありません」と出る

`config.json` を手で編集した際に、カンマの付け忘れ・付けすぎなどでファイルが壊れています。URLを追加するときは、以下のように**URLとURLの間にカンマ `,` を入れる**のを忘れないでください（一番最後のURLの後ろにはカンマを付けません）。

```json
"video_urls": [
  "URL1",
  "URL2"
],
```

### Q. 「Claude APIキーが設定されていないため、企画案の生成をスキップします」と出る

`config.json` の `anthropic_api_key` がまだ入力前のまま（プレースホルダーのまま）です。準備1の手順でキーを設定してください。データ収集・傾向分析自体は行われるので、Excelレポート自体は作られます（「投稿企画案」シートだけ空になります）。

### Q. 「投稿データが1件も取得できなかったため、企画案の生成をスキップします」と出る

`video_urls` が空っぽか、登録した動画URLが正しく取得できなかった場合に出ます。手順1〜2をやり直し、URLが正しくコピーされているか確認してください。

### Q. 特定の動画だけ「〇〇の取得に失敗しました」と出る

その1件だけスキップされて、残りの動画の処理は続けられます（レポート自体は問題なく作られます）。URLが間違っている、動画が削除された、TikTok側で一時的にブロックされた、などが考えられます。気になる場合はそのURLだけ後で単独で試してみてください。

### Q. 実行するとたくさん英語のエラー（Traceback）が出て止まる

上記の想定内エラー以外の問題が起きています。エラーメッセージの内容をコピーして、次の作業依頼のときに一緒に伝えてください。

## 5. 覚えておいてほしいこと

- **検索は自分でやる必要があります**: 「駄菓子」で自動検索してくれるわけではなく、あなたが見つけたURLをツールに渡す形です（TikTok側の仕組みで自動検索がブロックされてしまうため、この方式にしています）。
- **実行するたびにAI利用料が少額発生します**: `anthropic_api_key`を設定してレポートを作るたびに、Claude APIの利用料が数円程度かかります。
- **URLはずっと`config.json`に残ります**: 分析したい動画が変わったら、古いURLを消して新しいURLに入れ替えてください。全部残したままにすると、毎回すべての動画を再取得することになります。
